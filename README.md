## 系统角色
> 不同角色将展示不同的界面，对应不同的操作权限和功能
* 1.普通用户
  * 创建社团
  * 修改密码
  * 修改个人信息
* 2.系统管理员
  * 管理系统所有用户
  * 管理系统所有社团
  * 修改密码
  * 修改个人信息
* 3.社团创建者
  * 创建社团
  * 添加人员加入社团
  * 修改密码
  * 修改个人信息
* 4.社团管理员
  * 发布社团活动
  * 修改密码
  * 修改个人信息
